# UmanIT PHP images

## Supported tags
* `5.6`
* `7.0`
* `7.1`
* `7.2`

## Presentation
Custom php-fpm images based on official alpine versions.

Additional PHP extensions are installed, listed below.

## Additional PHP extensions
* bcmath
* iconv
* imagick
* intl
* mbstring
* mcrypt (except for PHP 7.2)
* opcache
* pgsql
* xsl
* zip
